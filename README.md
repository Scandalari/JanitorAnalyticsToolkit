Janitor Analytics Tool
by Scandalari

---

This tool uses data pulled from my Chrome extension (found here: https://chromewebstore.google.com/detail/janitor-analytics-helper/pkmmbkefnanefhaihfmgdmomegncidjo) to plot your growth as a creator on Janitor.

It lets you plot by tag as well, if you wanna see how well specific niches are performing for you.

OH! There's also some hidden easter eggs. See if you can find them. (Hint: Right after tutorial island, you'll find one of them.)

---

To install, Download JanitorAnalytics-Setup.exe from the latest Release.
Run it. Click "More Info" -> "Run Anyway" if Windows gets snippy.
Clicky through the wizard. Blammo. Done.
(Seriously, though. This is useless without my extension. Link above.)
